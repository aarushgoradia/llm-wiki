## Overview

The VC707 is a Xilinx Virtex-7 FPGA evaluation board, commonly used as glue logic and I/O bridging in research chip demo systems.

## Architecture / Design

Virtex-7 XC7VX485T FPGA with PCIe endpoint support and high-speed I/O.

## Key Properties

- In test setups it typically performs no computation — it serves as a protocol bridge between a host and a research chip.

## Papers Using This System

- [[2017-chen-eyeriss]] — serves as the PCIe controller between the [[jetson-tk1]] host and the Eyeriss chip in the demo system; performs no processing.
